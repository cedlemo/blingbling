append_function_get_events_from (calendar, my_function)
Add new function in order to get events from external application This method let the taskwarrior and remind links intact and add your founction

Parameters

  • calendar: a calendar
  • my_function: a function that you write

Usage:

my_cal:append_function_get_events_from(function(day, month, year) s="third function ".. " " .. day .. " " .. month .." " ..year return s end) This function display in the menu the string "third function 26 11 2011" for example.
